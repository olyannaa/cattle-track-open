# Cattle Track Frontend

React/Vite frontend for Cattle Track.

Основная документация находится в [корневом README](../README.md). Не храните реальные логины, пароли, токены и идентификаторы организаций в README или исходном коде.

## Local Run

```bash
npm ci
VITE_API_URL=http://localhost:5001/api/ npm run dev -- --host 0.0.0.0
```
