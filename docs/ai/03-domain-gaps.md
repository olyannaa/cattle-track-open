# Пробелы домена и roadmap (не инструменты)

То, что реально нужно фермеру, но не выражается тулом на текущей системе — потому что данных/функционала нет в API. Это НЕ тулы. Назначение документа: (1) продуктовая глубина на защите, (2) честный список ограничений, (3) материал для мотивационного письма (живые формулировки).

## Отсутствует в системе — потенциальные тулы после доработки backend

| Что нужно | Роль | Чего не хватает в системе |
|---|---|---|
| Сводка/дневной журнал событий (агрегатор по всем модулям) | управляющий | нет агрегирующего эндпоинта |
| Список активных задач/просрочек (кому пора вакцинация, у кого задержка отёла) | управляющий, ветврач | нет логики «что просрочено» |
| Групповой СУП/привес по группе за период | зоотехник | backend считает СУП только по одному животному |
| Аналитика с разбивкой по группе/типу | все | `StatisticsChartsController` фильтрует только по датам |
| Проверка срока вывода препарата перед продажей | ветврач | данные препарата есть, расчёта нет |
| Групповая/массовая обработка при вспышке | ветврач | API оперирует явным списком животных, не «всей группой» |
| Себестоимость привеса / экономика по группе | экономист | нет связки кормление+вес+рынок |
| Подбор пар без инбридинга | зоотехник | родословная есть, расчёта коэффициента инбридинга нет |

## За пределами системы учёта (не решается тулами — направления развития)

- Карантин при особо опасной инфекции (нет статуса «карантин», нет блокировки перемещений).
- Интеграция с Меркурий/Хорриот (данные готовятся, интеграции нет).
- Офлайн-режим для дальних пастбищ без связи.
- Геолокация/GPS стада.
- Отдельные причины выбытия: хищник, кража, стихийное бедствие (сейчас только общий «падёж» с текстом).
- Трудные роды (кесарево, выпадение матки), телёнок-сирота (каскад отёла не предусматривает).
- HR/наряды персонала.

## Использование на защите
Показать как доказательство понимания домена и границ, а не как обязательство реализовать. Формула: «реализовано и измерено (`00`/`01`/`02`) → следующий шаг (этот документ) → осознанно вне скоупа и почему». Честность о границах систематически ценится комиссией.
